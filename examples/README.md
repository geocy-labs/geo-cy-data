# Examples

`fermat_quartic_smoke.yaml` captures the baseline Fermat smoke-test parameters.

`cefalu_quartic_lambda_0_75.yaml` and `cefalu_quartic_lambda_1_0.yaml` capture two parameter choices for the Cefalu family benchmark.

The CLI currently takes flags directly, but keeping a named example configuration in the repository makes the benchmark settings easy to reference in documentation and future workflow automation.
