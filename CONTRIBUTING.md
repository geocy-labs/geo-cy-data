# Contributing

Thanks for contributing to GeoCYData.

## Development setup

```bash
git clone https://github.com/geocy-labs/geo-cy-data.git
cd geo-cy-data
pip install -e .[dev]
```

## Quality checks

```bash
pytest
ruff check .
ruff format .
```

## Scope

This repository is a research-software project. Favor small, reviewable changes, clear public APIs, reproducible examples, and direct documentation updates when behavior changes.
